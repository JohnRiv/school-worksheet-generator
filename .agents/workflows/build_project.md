---
description: Build the project by installing dependencies and running the build script
---

Follow these steps to build the School Worksheet Generator project:

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run TypeScript type checking** (Optional but recommended):
    ```bash
    npm run typecheck
    ```

3.  **Run linting** (Optional but recommended):
    ```bash
    npm run lint
    ```

4.  **Build the production bundle**:
    ```bash
    npm run build
    ```
