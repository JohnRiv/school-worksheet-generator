---
description: Start the development server and preview the application
---

Follow these steps to start the development server and preview the application:

1.  **Start the development server**:
    ```bash
    npm run dev
    ```
    This will start the server on [http://localhost:3000](http://localhost:3000) by default (or the next available port). Look for the console output to confirm the exact address.

2.  **Verify the page loads**: 
    Open the server URL in your browser and ensure the welcome screen with the "School Worksheet Generator" title is displayed.

3.  **Basic Interaction Testing**:
    -   Click the "Take Photo" button (requires camera access).
    -   Try dragging or selecting an image for upload.
    -   Toggle the "Attempt to remove handwriting" checkbox.
